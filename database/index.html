PERINTAH MYSQL BERDASARKAN DDL dan DML dalam SERVER BASISDATA MYSQL

A. DDL (Data Definition Language)

Merupakan perintah-perintah yang biasa digunakan administrator database untuk mendefinisikan skema dan subskema database.

PERINTAH DEFINISI CONTOH
CREATE Perintah ini digunakan untuk membuat, termasuk di antaranya membuat database baru, tabel baru, view baru, dan kolom. CREATE TABLE mahasiswa (
nama_mahasiswa varchar(225) default NULL,
nama_jurusan varchar(20) default NULL,
NIM varchar(10) default NULL,
PRIMARY KEY (nama_mahasiswa)
)
ALTER Perintah ini digunakan untuk mengubah struktur tabel yang telah dibuat. alter table akademik add email varchar (25);
TRUNCATE Menghapus semua isi dari suatu table database. TRUNCATE TABLE table_name;
DROP Perintah ini digunakan untuk menghapus database dan tabel. Drop table mahasiswa;
DESC/DESCRIBE Perintah Untuk melihat struktur table secara detail. DESC mahasiswa;
RENAME Perintah untuk mengganti nama table/ relasi. Rename table nama_table to nama_tabel baru;


B. DML (Data Manipulation Language)

Merupakan perintah-perintah yang biasa digunakan administrator database untuk mendefinisikan skema dan subskema database.Merupakan sub perintah dari bahasa SQL yang digunakan untuk memanipulasi data dalam database yang telah dibuat.

PERINTAH DEFINISI CONTOH
INSERT Perintah ini digunakan untuk menyisipkan atau memasukkan data baru ke dalam tabel. Penggunaannya setelah database dan tabel selesai dibuat. INSERT INTO mahasiswa (nim,nama,kelas) VALUES (‘L200170118’,'SARTIKA RIZKY M’,’B’);
UPDATE Perintah ini digunakan untuk memperbarui data lama menjadi data terkini. update mahasiswa set nama = 'YAYAS'
where npm = 'L20017009',;
DELETE Perintah ini digunakan untuk menghapus data dari table. DELETE FROM mahasiswa WHERE nama = ‘SARTIKA RIZKY M’;
SELECT Perintah ini digunakan untuk mengambil data atau menampilkan data dari satu tabel atau beberapa tabel dalam relasi. a. SELECT nama,nim, jurusan FROM mahasiswa;
b. SELECT * FROM mahasiswa;
c. SELECT nama,nim, jurusan FROM mahasiswa WHERE jurusan = ‘Informatika’;
d. SELECT * FROM mahasiswa WHERE nama like ‘AN%’ ORDER BY nama ASC;
